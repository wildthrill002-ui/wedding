CLAUDE.md - как работать в этом проекте

Правила для агента (обязательно к соблюдению):

- Объясняй всё максимально просто, без технического жаргона. Пользователь не разработчик и не обязан понимать термины — переводи их на простой язык.
- Перед удалением файлов, публикацией, отправкой куда-либо, деплоем или любым необратимым действием — всегда сначала спроси подтверждение.
- Никогда не клади секреты (.env, токены, пароли) в Git — проверяй .gitignore перед каждым коммитом.
- Когда пользователь пишет "закрой сессию" или "запушь в git" — сделай git add, git commit с понятным сообщением на русском, и git push в origin. Кратко перечисли, что изменилось.
- В конце сессии дай короткий итог простыми словами: что сделано, что изменилось, что дальше.
